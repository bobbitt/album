Album Photo - Les fichiers static HTML sont genérés
Ce mode est prévu pour permettre l'utilisation Off-line de votre album. Par exemple si le server est hors service ou si vous voulez graver votre album sur CD-ROM.
Ceci va créer un fichier HTML dans chaque répertoire (album), et un pour chaque photo. Ceci vous permettra la visualisation des photos via static HTML. (Normalement cet HTML est créer à la volée par le server web). L'inconvenient est que vous devez repeter cette operation à chaque fois que des photos sont ajoutées.
